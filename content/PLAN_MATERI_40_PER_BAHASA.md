# Rencana Konten ±40 Lesson per Bahasa

## Batasan
- Tidak menyalin mentah seluruh W3Schools.
- Materi yang dibuat: W3Schools-like (ringkas, banyak contoh, latihan), fokus konsep inti HTML/CSS/JS.

## HTML (target ~40 lesson)
1. Pengenalan & dasar dokumen (doctype, html lang, meta viewport, title)
2. Headings (h1-h6), p, hr, br
3. Struktur & elemen semantik (header, nav, main, section, article, aside, footer)
4. Komentar & organisasi kode
5. Links (a, href, target, rel)
6. Images (img, src, alt, figure/figcaption)
7. Lists (ul/ol/li, nesting)
8. Tables (table, thead/tbody/tfoot, caption, colspan/rowspan)
9. Forms (form, label, input, textarea, select, option)
10. Input types (text, email, number, password, date, checkbox, radio, range, file)
11. Attributes penting (id, class, name, value, required, minlength, pattern)
12. Form validation (novalidate, constraint validation)
13. Buttons & submit/reset
14. Media (audio, video, controls, source, poster)
15. Iframe dasar + security catatan (sandbox, referrerpolicy)
16. Special elements (div/span, strong/em, code/pre, kbd, samp)
17. Layout dasar (grid ringan via HTML: semantic wrappers)
18. Accessibility basics (alt, label for/id, semantic landmarks)
19. Meta tags dasar (charset, description, og basic)
20. HTML character entities
21. Path relative vs absolute (href, src)
22. Custom data attribute (data-*)
23. Responsive images basics (srcset concept - ringkas)
24. Progressive enhancement (konsep singkat)
25. Praktik mini: bikin landing sederhana (dalam beberapa lesson)

## CSS (target ~40 lesson)
1. Pengenalan CSS & cascade/spesifisitas
2. Syntax: selector, property, value, units
3. Inline/internal/external
4. Selector element/class/id
5. Selector kombinasi (descendant/child)
6. Selector pseudo-class (:hover, :focus)
7. Attribute selector [type="..."]
8. Box model lengkap + border-radius
9. Margin collapsing concept (ringkas)
10. Padding shorthand
11. Sizing: width/height, min/max
12. Typography (font, line-height, letter-spacing)
13. Text styling (color, background-clip concept ringkas)
14. Links styling (a:visited, a:hover)
15. Backgrounds (color, image, repeat/size)
16. Gradients
17. Flexbox: container properties
18. Flexbox: item properties (order, grow, shrink)
19. Flexbox: alignment & wrapping
20. CSS Grid: grid-template
21. Grid: placement (grid-column/row)
22. Grid: auto-fit/auto-fill (ringkas)
23. Positioning (static/relative/absolute/fixed/sticky)
24. Z-index & stacking context (ringkas)
25. Overflow & scroll
26. Transitions
27. Animations (@keyframes)
28. Transform (translate/scale/rotate)
29. Responsive: media queries
30. Responsive: fluid typography (ringkas)
31. Responsive: layout switching
32. Utility classes konsep
33. Practice: card UI
34. Practice: responsive navbar
35. Practice: gallery grid
36. Practice: modal sederhana (CSS)
37. Practical review (ringkas)

## JS (target ~40 lesson)
1. Pengenalan JS & runtime di browser
2. Values & data types
3. let/const/var (concept)
4. Operators: aritmatika/perbandingan
5. Truthy/falsy (konsep)
6. if/else
7. switch
8. for/while/do-while
9. for...of/for...in (ringkas)
10. Function declaration vs expression
11. Arrow function
12. Parameters & return
13. Scope & hoisting (konsep)
14. Array: map/filter/reduce (ringkas)
15. Array: find/findIndex/some/every
16. Destructuring (ringkas)
17. Spread/rest (ringkas)
18. Objects: literal & access
19. DOM: querySelector, getElementById
20. DOM: createElement, appendChild
21. DOM: event listeners
22. Events: click/input/change/submit
23. Form handling JS
24. Fetch API + async/await (konsep)
25. Errors: try/catch
26. Basic localStorage/sessionStorage
27. JSON stringify/parse
28. Modules konsep (ringkas)
29. Practice: todo list
