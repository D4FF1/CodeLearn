Belajar CSS

Kuasai styling web: selectors, box model, flexbox, grid, animasi, dan responsive design.

1

Pengenalan CSS & Spesifisitas

CSS (Cascading Style Sheets) mengatur tampilan website. Konsep cascade dan spesifisitas menentukan rule mana yang diterapkan jika terjadi konflik. Spesifisitas: inline style > ID > class/attribute > element. !important mengalahkan semuanya (hindari jika tidak perlu).

CSS 
/* Spesifisitas: ID (100) > class (10) > element (1) */
p { color: blue; }          /* specificity 1 */
.text { color: green; }       /* specificity 10 */
#judul { color: red; }        /* specificity 100 */
p.text#judul { color: purple; }  /* 1+10+100 = 111 */
Aturan Prioritas:
  • Inline style = 1000
  • ID selector = 100
  • Class, attribute, pseudo-class = 10
  • Element, pseudo-element = 1
2

Sintaks CSS: Selector, Property, Value, Units

Aturan CSS terdiri dari selector (elemen yang dituju), property (aspek yang diubah), dan value (nilai baru). Units: px (absolut), rem (relatif root), em (relatif parent), %, vw/vh.

CSS
h1 {
  font-size: 2rem;      /* 2x ukuran font root */
  margin: 1em;         /* 1x font-size element */
  width: 80%;          /* 80% dari parent */
  height: 100vh;       /* 100% viewport height */
}
Rekomendasi: Gunakan rem untuk font-size, px untuk border/shadow, % atau vw untuk layout lebar.
3

Inline, Internal, External CSS

Ada 3 cara menambahkan CSS ke HTML. External (file terpisah) adalah best practice.

HTML
<!-- External (disarankan) -->
<link rel="stylesheet" href="style.css">

<!-- Internal (di dalam <head>) -->
<style>
  body { background: #fff; }
</style>

<!-- Inline (hanya untuk kasus khusus) -->
<p style="color: red;">Teks merah</p>
External CSS lebih mudah maintenance, caching, dan reusable di banyak halaman.
4

Selector Element, Class, ID

Element selector menarget semua tag HTML. Class (.nama) bisa dipakai banyak elemen. ID (#nama) hanya satu per halaman.

CSS
p { margin-bottom: 1em; }
.button { background: blue; }
#header { position: fixed; }
.btn.primary { background: green; }  /* elemen punya kedua class */
5

Kombinasi Selector

div p (descendant) semua p di dalam div. div > p (child) hanya p langsung. h2 + p (adjacent) p tepat setelah h2. h2 ~ p (general sibling) semua p setelah h2.

CSS
article p { line-height: 1.6; }
ul > li { list-style: none; }
h2 + p { font-weight: bold; }
6

Pseudo-class (:hover, :focus, :nth-child)

Pseudo-class menarget elemen dalam keadaan tertentu. Contoh: :hover saat mouse di atas, :focus saat input aktif, :first-child, :nth-child(2).

CSS
a:hover { color: red; }
input:focus { outline: 2px solid blue; }
li:nth-child(odd) { background: #f0f0f0; }
7

Attribute Selector [attr="value"]

Memilih elemen berdasarkan atribut: [type="text"], [href^="https"] (awalan), [href$=".pdf"] (akhiran).

CSS
input[type="email"] { border-color: blue; }
a[href$=".pdf"]::after { content: " (PDF)"; }
8

Box Model Lengkap + Border Radius

Setiap elemen adalah box dengan: content, padding, border, margin. box-sizing: border-box membuat width/height termasuk padding dan border (sangat direkomendasikan).

CSS
.box {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
  border-radius: 12px;
  box-sizing: border-box;
}
Selalu gunakan box-sizing: border-box secara global: * { box-sizing: border-box; }
9

Margin Collapsing (konsep)

Margin vertikal antar elemen bisa bergabung (collapse) menjadi margin terbesar. Solusi: gunakan overflow: hidden atau padding/border pada parent, atau pakai flex/grid gap.

CSS
/* Collapsing terjadi pada: */
/* 1. Adjacent siblings */
.box1 { margin-bottom: 20px; }
.box2 { margin-top: 10px; }  /* jarak = 20px, bukan 30px */

/* Cara mengatasi: gunakan flex/gap atau padding pada parent */
10

Shorthand Padding & Margin

CSS
/* 4 nilai: top right bottom left (searah jarum jam) */
.card { margin: 10px 20px 10px 20px; }
/* 3 nilai: top (left-right) bottom */
.card { padding: 10px 20px 10px; }
/* 2 nilai: top-bottom left-right */
.card { margin: 10px 20px; }
/* 1 nilai: semua sisi */
.card { padding: 16px; }

17

Flexbox: Container Properties

Flexbox untuk layout satu dimensi. justify-content mengatur sumbu utama, align-items sumbu silang.

CSS
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
}
18

Flexbox Items (flex-grow, shrink, order)

CSS
.item { flex: 1; }  /* grow 1 */
.sidebar { flex: 0 0 250px; }
.order-2 { order: 2; }
19

Flexbox: align-self, flex-wrap

CSS
.container { flex-wrap: wrap; gap: 16px; }
.special { align-self: flex-end; }
20

CSS Grid: grid-template-columns/rows

Grid untuk layout 2 dimensi (baris DAN kolom).

CSS

  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
21

Grid Placement: grid-column, grid-row

CSS

22

Responsive: Media Queries

Mobile-first: tulis CSS untuk mobile dulu, lalu tambah rules untuk layar lebih besar dengan min-width.

CSS
.container { width: 100%; }
@media (min-width: 768px) {
  .container { width: 720px; margin: 0 auto; }
}
30

Fluid Typography: clamp()

CSS
h1 { font-size: clamp(1.5rem, 5vw, 3rem); }
p { font-size: clamp(1rem, 2vw, 1.25rem); }

37

CSS Review: Best Practices

Rangkuman konsep CSS penting yang harus diingat.

CSS
/* Global box-sizing */
* { box-sizing: border-box; }

/* Flexbox centering */
.center { display: flex; justify-content: center; align-items: center; }

/* Responsive grid */
.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); }

/* Custom properties (CSS variables) */
:root { --primary: #00d9a0; }
.btn { background: var(--primary); }
Yang selalu diingat:
  • Gunakan box-sizing: border-box secara global
  • Mobile-first: min-width media queries
  • Flexbox untuk 1 dimensi, Grid untuk 2 dimensi
  • Hindari !important sebisa mungkin
